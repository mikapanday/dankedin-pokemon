## Image Format

- Supported formats: PNG, JPG, JPEG, WEBP
- Recommended size: Square images work best (e.g., 200x200px or 400x400px)
- The images will be displayed as background images and will be scaled to fit

## Notes

- Make sure the file names match exactly (case-sensitive)
- If an image is missing, the game will display a placeholder
- These images represent people/applicants from the user's point of view
